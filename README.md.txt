Projeto NetBeans - SistemaNutricionistasPI

Como abrir:
1. Abra o NetBeans.
2. Vá em File > Open Project / Arquivo > Abrir Projeto.
3. Selecione a pasta SistemaNutricionistasPI.
4. Execute a classe Principal.java.

Protótipo implementado:
- Cadastro simulado de alimentos.
- Cadastro simulado de cardápio.
- Geração de pedido com cálculo automático de per capita.

Observação:
O projeto foi planejado para uso futuro com MySQL, conforme solicitado na atividade.
Nesta etapa, o protótipo usa dados em memória no método main().
